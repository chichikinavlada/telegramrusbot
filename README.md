# Telegram-бот для анализа русскоязычного учебного текста

Это учебный проект по Python. Я сделала телеграм-бота, который принимает кусок учебного текста и выдает по нему небольшой разбор.

## Что делает бот

Бот умеет:

 считать количество слов и предложений;
 находить самые частотные леммы;
 выделять сущности из текста;
 сохранять результаты в базу данных;
 строить простой график.

Под учебным текстом тут имеется в виду, например, кусок лекции, конспекта или статьи.

## Что использовано в проекте

Из NLP:

 `pymorphy3` для лемматизации;
 `natasha` для выделения сущностей.

Из других тем курса:
 `SQLite` для хранения истории анализов;
 `matplotlib` для графика;
 `aiogram` для самого телеграм-бота.

## Как это работает

Пользователь открывает бота и выбирает действие кнопками. Потом отправляет текст. После этого бот обрабатывает сообщение, считает статистику, выделяет леммы и сущности, сохраняет результат и отправляет ответ пользователю. Отдельно еще строится график по тексту.

## Версия Python

Проект делался на Python 3.10.

## Установка

Нужно установить зависимости:

`python -m pip install -r requirements.txt`

## Запуск

Точка входа: [main.py]

Запуск проекта:

`python main.py`

## Как пользоваться

В боте все работает через кнопки.

 «Разобрать текст» — отправить текст на анализ;
 «История» — посмотреть последние анализы;
 «Статистика» — посмотреть свою статистику.

Также можно просто отправить текст сообщением.

## Деплой
При деплое нужно добавить переменную окружения `BOT_TOKEN`.

Надо учитывать, что на бесплатном хостинге SQLite может работать нестабильно, поэтому история иногда может сбрасываться.
