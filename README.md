# Telegram-бот для анализа русскоязычного учебного текста

Небольшой учебный проект на Python: Telegram-бот принимает русскоязычный учебный текст и возвращает краткий лингвистический анализ.

## Что умеет бот

- считает количество слов и предложений;
- лемматизирует слова и показывает самые частотные леммы;
- выделяет именованные сущности (люди, места, организации);
- сохраняет историю анализов пользователя в SQLite;
- строит простую визуализацию результатов.

## Как проект закрывает требования

### Основная часть

- **NLP-методы:**
  - морфологический анализ и лемматизация через `pymorphy3`;
  - извлечение именованных сущностей через `natasha`.
- **Другие темы курса:**
  - база данных SQLite для хранения результатов;
  - визуализация результатов через `matplotlib`.

### Пользовательская часть

- интерфейс в виде Telegram-бота на `aiogram`.

## Python

Проект рассчитан на **Python 3.10+**.

## Установка

1. Используйте глобальный интерпретатор Python 3.10+.
2. Установите зависимости в глобальный Python:

  `python -m pip install -r requirements.txt`

3. Токен бота сейчас прописан прямо в [src/config.py](src/config.py).

  При желании можно поменять его там же.

  Если проект запускается на хостинге, можно передать токен через переменную окружения `BOT_TOKEN`.

## Запуск

Точка входа в проект: [main.py](main.py)

Запуск:

`python main.py`

## Как пользоваться ботом

- все управление идет через inline-кнопки;
- кнопка «Разобрать текст» переводит бота в режим анализа;
- кнопка «История» показывает последние анализы;
- кнопка «Статистика» показывает общую статистику пользователя.

Также можно просто отправить текст сообщением.

## Деплой

В проекте есть [render.yaml](render.yaml) для простого деплоя на Render как background worker.

Нужно:
- загрузить проект в GitHub;
- создать на Render новый Blueprint из репозитория;
- Render сам прочитает [render.yaml](render.yaml);
- в переменных окружения указать `BOT_TOKEN`;
- после деплоя бот запустится как worker.

Важно: SQLite на бесплатном хостинге обычно хранится не очень надежно, поэтому история анализов может сбрасываться после перезапуска сервиса.
